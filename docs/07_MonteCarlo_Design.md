# Monte Carlo Simulation Design

## Objective

Estimate potential future portfolio outcomes using probabilistic simulation.

The simulation helps investors understand the range of possible future portfolio values rather than relying on a single forecast.

---

# Inputs

Required Inputs:

* Initial Investment Amount
* Expected Annual Return
* Annual Volatility
* Investment Horizon
* Number of Simulations

---

# Example Inputs

Initial Investment:

$10,00,000

Expected Return:

10%

Volatility:

15%

Investment Horizon:

20 Years

Simulations:

10,000
